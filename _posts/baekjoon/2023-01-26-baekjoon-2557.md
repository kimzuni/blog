각 언어로 출력하는 방법만 알면 풀 수 있다.

## Bash
```bash
echo "Hello World!"
```

## C
```c
#include <stdio.h>

int main(void) {
	printf("Hello World!\n");
	return 0;
}
```

## Node.js
```javascript
console.log("Hello World!");
```
`Node.js`는 언어가 아니라 `JavaScript` 런타임 환경이다.  
즉, `Node.js`는 `JavaScript` 문법으로 작성하면 된다.

## PHP
```php
Hello World!
```
`HTML`에서 `PHP`를 사용하는 것처럼 `<?php ... ?>` 태그 외부에 있는 내용은 그대로 출력한다.

## Python3
```python
print("Hello World!")
```

## Ruby
```ruby
puts "Hello World!"
```
