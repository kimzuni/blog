두 수의 차이인 543을 빼면 된다.

## Bash
```bash
read year
echo $((year - 543))
```

## C
```c
#include <stdio.h>

int main(void) {
	int year;
	scanf("%d", &year);
	printf("%d\n", year-543);

	return 0;
}
```

## Node.js
```javascript
let input = require("fs").readFileSync("/dev/stdin").toString().split("\n");
console.log(parseInt(input[0]) - 543);
```

## Python3
```python
print(int(input()) - 543)
```

## Ruby
```ruby
puts gets.chomp.to_i - 543
```
