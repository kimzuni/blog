입력한 값이 `0 0`이면 반복문을 빠져나온다.

## Bash
```bash
while :; do
	read a b
	if [ "$a$b" == "00" ]; then
		break
	fi
	echo $((a + b))
done
```
`while :` == `while true` == 무한루프

## C
```c
#include <stdio.h>

int main(void) {
	int a, b;
	while (1) {
		scanf("%d %d", &a, &b);
		if (a == 0 && b == 0) { break; }
		printf("%d\n", a+b);
	}
	return 0;
}
```

## PHP

## Python3
```python
while 1:
    a, b = map(int, input().split())
    if a==0 and b==0: break
    print(a+b)
```

## Ruby
```ruby

  a, b = gets.chomp.split().map {|i| i.to_i}
  break if a==0 and b==0

end
```
