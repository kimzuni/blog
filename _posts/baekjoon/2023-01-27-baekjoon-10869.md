## Bash
```bash
read a b
echo $((a + b))
echo $((a - b))
echo $((a * b))
echo $((a / b))
echo $((a % b))
```

## C
```c
#include <stdio.h>

int main(void) {
	int a, b;
	scanf("%d %d", &a, &b);
	printf("%d\n", a + b);
	printf("%d\n", a - b);
	printf("%d\n", a * b);
	printf("%d\n", a / b);
	printf("%d\n", a % b);
	return 0;
}
```

## Node.js
```javascript
let [a, b] = require("fs").readFileSync(0).toString().trim().split(" ").map(Number);
console.log(a+b);
console.log(a-b);
console.log(a*b);
console.log(Math.floor(a/b));
console.log(a%b);
```
`Math.floor` 대신 `parseInt`를 사용해도 된다.

## PHP
```php
<?php
	fscanf(STDIN, "%d %d", $a, $b);
	echo $a + $b."\n";
	echo $a - $b."\n";
	echo $a * $b."\n";
	echo floor($a / $b)."\n";
	echo $a % $b."\n";
?>
```

## Python3
```python

print(a-b)
```
