## Bash
```bash
read a b
echo $(($a * $b))
```

## C
```c
#include <stdio.h>

int main(void) {
	int a, b;
	scanf("%d %d", &a, &b);
	printf("%d", a*b);
	return 0;
}
```

## Node.js
```javascript
let input = require("fs").readFileSync("/dev/stdin").toString().split(" ").map(x => parseInt(x));
console.log(input[0] * input[1]);
```

## Python3
```python
a, b = map(int, input().split())
print(a*b)
```

## Ruby
```ruby
a, b = gets.chomp.split().map {|i| i.to_i}
puts a*b
```
