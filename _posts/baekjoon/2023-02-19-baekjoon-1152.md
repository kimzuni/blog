문자열을 공백을 기준으로 자른 후 나온 단어의 개수는?

## Bash
```bash
read -a arr
echo ${#arr[@]}
```

## Node.js
```javascript
const arr = require("fs").readFileSync(0).toString().trim().split(" ");
console.log(arr.length);
```
왜 틀린 건지 모르겠다..

## Python3
```python
print( len(input().split()) )
```

## Ruby
```ruby
puts gets.chomp.split().size
```
