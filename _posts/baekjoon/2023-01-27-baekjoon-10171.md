귀여운 고냥이.  
역슬래시와 같이 특별한 기능으로 사용되는 문자를 출력할 때 주의해야 한다.

## Bash
```bash
echo "\    /\\
 )  ( ')
(  /  )
 \(__)|"
```

## C
```c
#include <stdio.h>

int main(void) {
	printf("\\    /\\\n");
	printf(" )  ( ')\n");
	printf("(  /  )\n");
	printf(" \\(__)|\n");
	return 0;
}
```

## Node.js
```javascript
console.log(`
\\    /\\
 )  ( ')
(  /  )
 \\(__)|
`.trim());
```

## PHP
```php
\    /\
 )  ( ')
(  /  )
 \(__)|
```

## Python3
```python
print("""
\    /\\
 )  ( ')
(  /  )
 \(__)|
""".strip())
```

## Ruby
```ruby
puts "
\\    /\\
 )  ( ')

```
