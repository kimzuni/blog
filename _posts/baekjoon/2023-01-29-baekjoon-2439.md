## Bash
```bash
read n
i=0
while [ $i -lt $n ]; do
	i=$((i + 1))
	j=$i
	o=""
	while [ 0 -lt $j ]; do
		o+=\*
		j=$((j - 1))
	done
	printf "%${n}s\n" "$o"
done
```
Bash에도 `printf` 명령어가 있다.

## C
```c
#include <stdio.h>

int main(void) {
	int n;
	scanf("%d", &n);
	for (int i=1; i<=n; i++) {
		for (int j=0; j<n-i; j++) {
			printf(" ");
		}
		for (int j=0; j<i; j++) {
			printf("*");
		}
		printf("\n");
	}
	return 0;
}
```

## Node.js

## Python3
```python
n = int(input())
for i in range(1, n+1):
    o = []
    for j in range(n):
        o.append(" " if j < n-i else "*")
    print("".join(o))
```
