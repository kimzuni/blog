## Bash
```bash
read a b
# expr $a + $b
# echo $[$a + $b]
echo $(($a + $b))
```
`expr`이 먹히지 않는다.  
3, 4번 라인 둘 다 정답이다.

## C
```c
#include <stdio.h>

int main(void) {
	int a, b;
	scanf("%d %d", &a, &b);
	printf("%d", a+b);
	return 0;
}
```

## Node.js
```javascript
let input = require("fs").readFileSync("/dev/stdin").toString().split(" ").map(x => parseInt(x));
console.log(input[0] + input[1]);
```

## Python3
```python
a, b = map(int, input().split())
print(a+b)
```

## Ruby
```ruby
a, b = gets.chomp.split().map {|i| i.to_i}
puts a+b
```
