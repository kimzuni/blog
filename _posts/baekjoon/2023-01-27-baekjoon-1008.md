## Bash
```bash
read a b
a=$((a * 10**10))
r=`echo $((a / b))`
n=0
if [ ${#r} -gt 10 ]; then
	n=${r::1}
	r=${r:1}
fi
echo $n.$r
```
Bash로 계산 시 소수부를 모두 버린다.  
그냥 아예 계산을 하지 않는 듯하다.  
그 예로 Bash에서 `3 / (3/2)`의 결과는 2가 아닌 3이 된다.

## C
```c
#include <stdio.h>

int main(void) {
	double a, b;
	scanf("%lf %lf", &a, &b);
	printf("%.9lf", a / b);
	return 0;
}
```
