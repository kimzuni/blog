## C
```c
#include <stdio.h>

int main(void) {
	double a, b;
	scanf("%lf %lf", &a, &b);
	printf("%.9lf", a / b);
	return 0;
}
```

## Node.js
```javascript
let input = require("fs").readFileSync("/dev/stdin").toString().split(" ").map(x => parseInt(x));
console.log(input[0] / input[1]);
```

## Python3
```python
a, b = map(int, input().split())
print(a/b)
```
