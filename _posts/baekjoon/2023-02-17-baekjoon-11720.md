주어진 수의 각 자릿수를 모두 더한 값은?

## Bash
```bash
read n
read m
#while [ $m != 0 ]; do
#	((t += m%10, m /= 10))
#done
for ((i=0; i<$n; i++)); do
	add=${m:$i:1}
	((t += add))
done
echo $t
```
주석 처리된 코드는 백준에서 틀린 답이라고 나온다.
