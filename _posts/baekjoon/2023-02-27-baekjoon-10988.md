기러기 토마토 스위스 인도인 별똥별 우영우 ..역삼역

## Bash
```bash
read str
rev=""
cnt=${#str}
while [ 0 -lt $cnt ]; do
	((cnt--))
	rev+=${str:$cnt:1}
done
test "$str" == "$rev" && echo 1 || echo 0
```

## Node.js
```javascript
const str = require("fs").readFileSync(0).toString().trim();
const rev = str.split("").reverse().join("");
const o = str == rev ? 1 : 0;
console.log(o);
```
